JavaScript

Язык сценариев JavaScript предназначен для создания интерактивных HTML-документов. Основой языка является понятие объекта.

DHTML

Динамический HTML (DHTML) является дальнейшим развитием языка HTML, добавляет к HTML набор свойств, позволяющих определять внешний вид документа и управлять слоями. При создании web-страницы с помощью DHTML у разработчика появляются дополнительные возможности.

CSS

CSS (англ. Cascading Style Sheets — каскадные таблицы стилей) — технология описания внешнего вида документа, написанного языком разметки. CSS используется создателями веб-страниц для задания цветов, шрифтов, расположения и других аспектов представления документа.
